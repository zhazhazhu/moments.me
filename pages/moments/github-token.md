# 申请 Github token

<p>github token 相当于你 github 的一把钥匙，而谁拥有这把钥匙就可以打开你 github 的大门，而你可以赋予这个钥匙权限，就是说你制作这个钥匙的时候指定这把钥匙可以干什么(登录/访问用户信息/修改仓库/等等...)</P>
