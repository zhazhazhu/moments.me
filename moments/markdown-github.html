[Github Api]在线编辑 Markdown 和 图床工具

我的初衷是方便我编写 部署在 GitHub 上的博客，然后项目的图片本地资源太大，这样就导致加载很慢，当然图床工具目前也有很多优秀的项目，例如 picx,可是他的压缩是默认压缩率，有时我想原图或者更自定义的压缩率，就遇到了问题

使用

  • 首页

  • 配置 github token 如何申请

现有功能

  • markdown 编辑
  • 多窗口操作
  • 新建 markdown
  • 新建文件夹
  • 删除文件
  • jsdelivr cdn 加速
  • 上传图片 (支持拖拽/浏览/粘贴 上传)
  • 自定义压缩率
  • 一键复制 [markdown/cdn] 链接
